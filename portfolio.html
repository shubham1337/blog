Personal Website

A fully responsive, Cross-browser compatible website. Hosted using Github Pages and generated using Jekyll. Icons used in this website are all SVG (Scalable Vector Graphics), for sharp and clean look, Agnostic of the screen they are viewed in (Even the triangles in the banner background are all SVG). My Development toolset for this website was as follows: Grunt, for building the website and watching for file changes, which eases the development cycle, Bower, Package manager for the front-end which aided in quick experimentation of some new packages, SASS for managing CSS better.
The source for this website is available here.

Mookit

A Massively Open Online Course Platform being developed at IIT Kanpur. It has been used to run online courses such as Architecture for Cloud and Mooc on Mooc which had a worldwide audience and massive number of registrations, Currently it is being used by IITK to conduct a series of MOOCs in the field of Agriculture named agMoocs. It was my privilege to be working with the diligent team at IITK. The team consisted of 4 developers, from which 2 worked at the back-end (which is written in NodeJS) and 2 (including me) at the front-end (EmberJS). Technologies that I had to tackle with on a day to day basis were: EmberJS, Handlebars, Socket.IO, jQuery, JavaScript, HTML, CSS and many more javascript libraries. It was a phenomenal experience for me and my first project which is being used by thousands of students to learn and grow.
